This is an ongoing Python script that aims to 2-way sync tasks from Todoist to Notion. 

Currently working on: 
- Making Todoist subtasks with parentId to sync with Notion, creating a relation between the task and the parent in Notion.

Not yet started:
- Updating tasks from Todoist to Notion. 
- Finding what tasks have been completed in Todoist and completing them in Notion. 
- Notion to Todoist Sync
